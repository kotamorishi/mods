# App Store Listing

## App Name
mods

## Subtitle
GitHub-Style Markdown Viewer

## Description
A fast, lightweight markdown viewer for macOS. No editing — just clean, beautiful rendering with full GitHub Flavored Markdown (GFM) support using the same engine GitHub uses (cmark-gfm).

**Rendering:**
- Full GitHub Flavored Markdown (GFM) compliance
- Syntax-highlighted code blocks (16+ languages)
- Tables, task lists, footnotes, strikethrough
- GitHub-style alerts ([!NOTE], [!WARNING], etc.)
- Math expressions (KaTeX) and Mermaid diagrams
- Emoji shortcodes and color chips for hex/rgb/hsl
- Dark mode / light mode with automatic switching

**Productivity:**
- Multiple windows with native tab support
- Find in document (Cmd+F) with live highlighting and search suggestions
- Keyword highlighting — set keywords to automatically highlight across all documents
- Table of Contents sidebar for heading navigation
- Pagination for large files with page navigation
- Zoom in/out (25%–500%) with slider and keyboard shortcuts
- Word count and reading time in status bar
- Copy button on code blocks
- Auto-reload on file changes (tracks renames too)
- Print (Cmd+P) and Export as PDF (Cmd+Shift+E)
- Custom CSS support for personalized styling

**Security & Privacy:**
- External images blocked by default — click to load, or whitelist trusted domains
- Trusted domains shared with QuickLook for automatic image loading
- No JavaScript execution from markdown content
- HTML sanitization and Content Security Policy
- No tracking, no analytics, no data collection

**Integration:**
- QuickLook preview in Finder (Space key) with full rendering
- Double-click .md files in Finder
- File > Open Recent
- Drag & drop support
- mods:// URL scheme for scripting
- Terminal: open -a mods file.md
